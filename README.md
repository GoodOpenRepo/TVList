# 电视节目单

### 介绍
能够播放各大卫视直播节目，并且支持节目回播。

### 播放器
使用[Vitamio](https://www.vitamio.org/)开源库封装的播放器，[Vitamio](https://www.vitamio.org/)能够流畅播放720P甚至1080P高清MKV，FLV，MP4，MOV，TS，RMVB等常见格式的视频，还可以在 Android 与 iOS 上跨平台支持 MMS, RTSP, RTMP, HLS(m3u8) 等常见的多种视频流媒体协议，包括点播与直播。
